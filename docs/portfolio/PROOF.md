## Python Calculator Application

### Summary

This is an interactive command-line calculator application built in Python 3.10. It uses a Read-Eval-Print Loop interface so users can enter commands and receive results for arithmetic operations, history viewing, and exit behavior.

### Portfolio Meaning

This project shows I have the foundational skills to build products without relying on an AI assistant. It was built before AI assistants became part of my normal workflow, which matters because it proves I understand what is happening underneath the application: command routing, input handling, error handling, state/history management, modular design, logging, and automated testing. It shows that I started building before the AI boom era and that my current AI work is built on real programming fundamentals.

### Features

- REPL command interface.
- Plugin architecture for arithmetic operations.
- Addition, subtraction, multiplication, and division commands.
- History management for previous calculations.
- Division-by-zero handling.
- Invalid input handling.
- Logging for events, errors, and debugging.
- Automated tests and coverage tooling.

### Tech Stack

- Python 3.10
- pytest
- pytest-cov
- Faker
- pandas
- logging
