# Anthony Quispilaya Portfolio

This repository contains Anthony Quispilaya's personal portfolio website.

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```

## Project Map

- `app/`: Next.js app route, layout, icon, and global styles.
- `components/`: portfolio-specific React components.
- `public/assets/`: portfolio images and proof assets.
- `docs/portfolio/`: portfolio planning notes, workshop answers, sprint plan, and proof/evidence notes.

`public/.nojekyll` is intentionally kept for GitHub Pages deployments so exported `_next` assets are served as static files.
